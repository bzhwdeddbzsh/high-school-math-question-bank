# 待校对中心

> 本页只记录真正需要人工介入的问题。不要把所有 AI 入库题目都默认加入这里，避免产生低价值待办。

## 使用原则

普通且能可靠转写的文字题，只需在题目元信息中标记 `status: 初步入库` 或 `status: 已转写`，不进入本页。

只有以下情况才加入本页：

1. OCR 识别不完整，例如题干缺字、公式明显异常、选项残缺。
2. 答案或详解缺失，或由 AI 自行补写且不确定。
3. 标签无法确定，或新增了标签体系中没有的标签。
4. 难度判断明显不确定。
5. 题目依赖图形，但图形无法可靠转成可编辑 Markdown/LaTeX/文字描述。
6. 立体几何、解析几何、函数图像、统计图表等图形题中，原图无法自动截取或无法准确重绘。
7. 原题在一整套大卷子 PDF 中，图片位置不好自动裁切，需要人工截图或补图。
8. Word 试卷生成时某题缺少必要图形，可能影响学生作答。

## 待办格式

每条待办必须包含：

- 待办类型：`题干校对` / `答案校对` / `标签确认` / `难度确认` / `图形补充` / `图形重绘` / `来源确认`
- 具体题目链接：例如 `[[2024 全国新课标 I 卷#第 16 题]]`
- 当前状态：待处理 / 已完成 / 暂缓
- 简短说明：说明为什么需要人工处理
- 创建日期
