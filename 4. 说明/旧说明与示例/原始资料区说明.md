# 原始资料区说明

本区用于保存所有尚未或已经转写的原始资料。**原始资料不要删除**，它们是题目溯源、核对图片、补充答案的重要依据。

## 子目录

- `PDF`：整套试卷、练习册、教辅扫描 PDF、高考真题 PDF。
- `Word`：可复制文本的 `.docx`、`.doc`、学校试卷、讲义。
- `截图`：单题截图、网页截图、聊天记录中的题目图片。
- `扫描件`：手机扫描、多页扫描图、拍照件。

## 命名建议

建议文件名包含：年份、地区/学校、考试类型、资料名。

示例：

- `2024-全国新课标I卷-数学.pdf`
- `2025-福建福州某校-高三上期中-数学.docx`
- `2026-07-10-零散单题-函数导数-截图01.png`

## 入库后如何链接原始资料

在题目中使用 Obsidian 内部链接保留溯源，例如：

```markdown
原始资料链接：[[1. 原始题目/PDF/2024-全国新课标I卷-数学.pdf]]
```

如果题目使用了截出的原图，可同时链接2. 题库/附件：

```markdown
题图：![[2. 题库/附件/截题图/2024-全国新课标I卷-第18题-立体几何.png]]
原始资料链接：[[1. 原始题目/PDF/2024-全国新课标I卷-数学.pdf]]
```
