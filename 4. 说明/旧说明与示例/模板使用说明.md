# 模板使用说明

本区保存题库入库、出卷、讲义需求的 Markdown 模板。初版不依赖 Obsidian 插件，复制模板内容即可使用。

## 模板列表

- [[3. 模版/整套试卷入库模板]]
- [[3. 模版/单题入库模板]]
- [[3. 模版/出卷需求模板]]
- [[3. 模版/讲义需求模板]]

## 使用方式

1. 整套试卷、整份练习：按 [[3. 模版/整套试卷入库模板]] 在 `2. 题库/文档化题库` 中新建笔记。
2. 零散单题：按 [[3. 模版/单题入库模板]] 追加到 `2. 题库/单题题库` 对应板块页面末尾。
3. 出卷：复制 [[3. 模版/出卷需求模板]] 到 [[5. 讲义与试卷/候选题清单]] 或聊天框中让 Codex/Claude 处理。
4. 讲义：复制 [[3. 模版/讲义需求模板]] 到 [[5. 讲义与试卷/讲义草稿]] 或聊天框中让 Codex/Claude 处理。
