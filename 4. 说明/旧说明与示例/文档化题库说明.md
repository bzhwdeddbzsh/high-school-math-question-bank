# 文档化题库说明

本区用于保存“一份资料一篇 Markdown 笔记”的题库文档。

适用资料：

- 一整套高考卷、模拟卷、校考卷。
- 一整份专题练习或讲义。
- 一份 Word 试卷或 PDF 试卷。

## 建库原则

1. 一份资料对应一篇 Markdown 笔记。
2. 笔记内部按题号拆分。
3. 每题必须使用独立标题：`## 第 1 题`、`## 第 2 题`。
4. 每题标签放在题目块上方。
5. 每题包含：标签、难度、来源、状态、题干、答案与详解、原始资料链接。
6. 答案与详解默认展开，必要时可使用 `<details>` 折叠版。
7. 所有原始 PDF、Word、截图、扫描件必须保留在 `1. 原始题目`，并登记到 [[1. 原始题目/来源登记]]。

## 推荐文件名

- `2024 全国新课标 I 卷.md`
- `2025 福建福州某校高三上期中数学.md`
- `函数与导数专题练习 01.md`
