# 单题汇总说明

本区用于保存“零散单题”的长期积累。

## 规则

1. 零散单题不需要每题单独开一篇笔记。
2. 按高中数学 7 大板块追加到对应汇总页。
3. 每道题必须有清楚标题，建议格式：`## 板块简称-单题-001`。
4. 每道题上方放标签，标签必须优先来自 [[4. 说明/高中数学三级标签体系]]。
5. 每道题包含：难度、来源、状态、题干、答案与详解、原始资料链接。
6. 若题目有图片，优先嵌入 `2. 题库/附件/截题图` 或 `2. 题库/附件/原图` 中的文件。
