# 高中数学题库

这是一个面向高中数学老师长期维护的 Obsidian 题库，主要服务中国大陆新高考体系，当前以新高考 I 卷风格为基准。

## 如何使用

1. 用 Obsidian 打开本仓库目录。
2. 从 `题库首页.md` 进入题库、模板、待校对中心和出卷工作流。
3. 新题入库、标签、难度、图片处理和出卷规则请优先阅读 `AGENTS.md` 与 `4. 说明/工作流/工作流总览.md`。

## 目录说明

- `1. 原始题目`：PDF、截图、扫描件等原始资料和来源登记。
- `2. 题库/文档化题库`：整套试卷或整份练习，一份资料一篇 Markdown。
- `2. 题库/单题题库`：零散单题按高中数学 7 大板块汇总。
- `2. 题库/附件`：题图、原图、截题图等附件。
- `3. 模版`：整卷、单题、出卷需求、讲义需求模板。
- `4. 说明`：标签体系、待校对中心、工作流说明和旧示例。
- `5. 讲义与试卷`：候选题清单、试卷与讲义生成说明。

## 共享提示

仓库中包含原始 PDF、截图和转载资料。建议使用私有仓库与指定同伴共享，并在使用时保留原始资料溯源链接。
