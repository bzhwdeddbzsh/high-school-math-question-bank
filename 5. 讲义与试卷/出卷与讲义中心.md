# 出卷与讲义中心

本区用于保存出卷候选题清单、生成试卷、讲义草稿等过程文件。

## 子目录

- `候选题清单`：Codex/Claude 根据需求筛选出的候选题，等待人工确认。
- `生成试卷`：已生成或待生成的试卷说明、最终 `.docx` 可放在 `2. 题库/附件/生成文档`。
- `讲义草稿`：专题讲义、例题讲义、复习讲义草稿。

## 出卷默认流程

1. 你提出出卷需求，最好使用 [[3. 模版/出卷需求模板]]。
2. Codex/Claude 根据标签、难度、来源、题量筛选候选题。
3. 先生成候选题清单，不直接生成 Word。
4. 你确认候选题后，Codex/Claude 再生成学生版 `.docx` 试卷。
5. 默认学生版不含答案解析。

详见：[[4. 说明/工作流/出卷与Word生成工作流]]

## 讲义默认流程

1. 使用 [[3. 模版/讲义需求模板]] 明确主题、标签、题量、难度梯度。
2. 先生成 Markdown 讲义草稿。
3. 需要时再转换为 Word 或 PDF。
