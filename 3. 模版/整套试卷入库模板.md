# 整套试卷入库模板

> 适用于：一整套试卷、一整份练习、一份校考卷、一份高考卷。使用方式：复制本模板到 `2. 题库/文档化题库` 中，文件名改为资料名称。

```yaml
type: 整套试卷入库
source_file: [[1. 原始题目/PDF/填写原始文件名.pdf]]
created: YYYY-MM-DD
status: 初步入库
```
